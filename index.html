What will be talked about?

Below is a list of topics that will be built upon:

Topic Description
Github Pages A site hosting service that takes HTML, CSS, and JavaScript files and publishes a website
HTML Formally known as Hypertext Markup Language. Is used to outline basic elements of a website
CSS Formally known as Cascading Style Sheets. Is used to format and style html elements.
JavaScript Formally known as JavaScript. Is used to give functionality to the html elements.

Where should I start?

Technically, you could start anywhere. If you have no knowledge of anything, you might as well start learning about github pages because it basically supports viewing html, css, and javascript all together in a published website. (Plus, it only takes a short amount of time to get a hang of the basics.)

Afterward, go ahead and learn html first because it sets the website structure that css and javascript will act upon. Once you've laid down the basic structure, then you can learn css and javascript.

To note: it generally helps to learn html, css, and javascript in conjunction. Doing so will teach you to coordinate all three of these things at the same time, which you'll have to do down the line if you want to make more websites. It just helps to have knowledge in making the basic structure first by using something like html in order to get started.
